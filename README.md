# Guarde-Software

**Guarde Software** es un sistema desarrollado para [**Guarde Lo Que Quiera**](https://www.guardeloquequiera.net/), empresa líder en el alquiler de bauleras y servicios de almacenamiento seguro para particulares y empresas, ubicada en Munro, Buenos Aires. Este proyecto consiste en una **aplicación web integral** diseñada para optimizar la gestión de clientes, bauleras y transacciones.  

## Tecnologías utilizadas  
- **SQL Server**: Gestión y almacenamiento de datos.  
- **ASP.NET**: Desarrollo del backend, lógica de negocio y comunicación con la base de datos.  
- **Angular**: Desarrollo del frontend, proporcionando una interfaz de usuario moderna, dinámica e intuitiva.  

## Beneficios para la empresa  
- **Centralización de la información**: Toda la gestión de clientes, pagos y contratos se concentra en un único sistema, eliminando la dependencia de Excel.  
- **Automatización de procesos**: La actualización de estados de cuenta, el control de disponibilidad de bauleras y el envío de notificaciones a los clientes se gestionan de forma automática, reduciendo tiempos y carga operativa.  
- **Reducción de errores**: El registro y validación de datos directamente en el sistema minimiza equivocaciones comunes en el ingreso de montos, clientes u operaciones.  
- **Interfaz moderna y amigable**: Basada en Angular, pensada para mejorar la experiencia del usuario y reducir el tiempo de capacitación del personal.  
- **Escalabilidad**: El sistema está diseñado para crecer con la empresa, adaptándose a sus necesidades y permitiendo la incorporación de futuras funcionalidades como facturación electrónica o reportes avanzados.  

## Descripción técnica  
El sistema se compone de tres capas principales:  
1. **Base de Datos (SQL Server):** estructura robusta para almacenar información de clientes, pagos y contratos de alquiler.  
2. **Backend (ASP.NET):** encargado de la lógica de negocio, seguridad y servicios de comunicación entre la base de datos y la aplicación.  
3. **Frontend (Angular):** interfaz moderna que facilita la interacción del usuario con el sistema de forma rápida y eficiente.
